1.Discuss the scope of `var`, `let`, and `const`


Answer: Absolutely, let's dive into the differences between `var`, `let`, and `const` in JavaScript. These are all ways to declare variables, but they behave differently in terms of scope, reassignment, and mutability.
*Var:- Var is a function scoped or global scope but not block scope.Var variable can be reassigned without any restriction. 

            #Example:
            function exampleVar() { 
                if (true) {
                    var x = 76;
                }
                console.log(x);
            }
                output = 76
**Let:- Let is a block scoped.Variables inside var can be updated and redeclared. But let can be updated but not redeclared. 

                #Example:
                function exampleLet() {
                if (true) {
                    let y = 10;
                }
                console.log(y); 
            }
                output: 'y' is not defined
***Const:- Const is a block scoped.Const cannot be updated or redeclared. Var and Let can be declared without initialization, const must be initialized when declared.Const variables cannot be redeclared or reassigned after their initial value has been assigned. 

            #Example:
            function exampleConst() {
                const y = 30;
                y = 40; 
                const y = 40
            }   
                output: 'y' is not defined 
                because 'y' is already declared.

Answer:API full form is application programming interface. An API is a set of rules that define how applications or devices can connect to and communicate with each other. A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style.
